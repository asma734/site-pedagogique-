🧠 Cours HTML - Niveau Débutant

1. Qu'est-ce que HTML ?

HTML (HyperText Markup Language) est le langage de base des pages web. Il permet de structurer les informations à afficher, comme des titres, des paragraphes, des images ou des liens.

2. Structure de base

<!DOCTYPE html>
<html>
  <head></head>
  <body>
    Contenu
  </body>
</html>

3. Liste des balises HTML utiles

    🧱 Structure de base d'une page HTML

  • <!DOCTYPE> : Déclare le type de document HTML (HTML5)
  • <html> : Conteneur principal de toute la page

  • <meta> : Déclare l'encodage (ex : UTF-8)

  • <body> : Contenu visible affiché dans la page

  • <span> : Conteneur en ligne (inline), utile pour appliquer un style

🧠 Cours CSS - Niveau Débutant

1. Qu'est-ce que CSS ?

Le CSS permet de styliser les pages HTML : couleurs, marges, polices, animations, etc. Il sépare le contenu (HTML) de la présentation (CSS).

🖍️ Syntaxe de base

sélecteur {
  propriété: valeur;
}

Exemple :

h1 {
  color: red;
  text-align: center;
}

📌 Méthodes d'intégration CSS

  • En ligne : directement dans la balise HTML →
    <p style="color:red;">Texte rouge</p>
  • Interne : à l'intérieur d'une balise <style> dans le <head> →
    <style>
  p {
    color: blue;
  }
</style>
  • Externe : via un fichier CSS séparé →
    <link rel="stylesheet" href="style.css"> on écrit cette ligne dans le code HTML pour faaire la liaison entre code html et le code CSS

    • 🏋️ Tableau récapitulatif

    MéthodeAvantagesInconvénients
    En ligneRapide à testerDifficile à maintenir
    InterneTout en un seul fichierPas réutilisable sur d'autres pages
    ExterneCentralisation des stylesFichier CSS externe requis

🧰 Flexbox (Flexible Box Layout)

Flexbox est un système moderne de mise en page qui permet d’aligner et de répartir facilement des éléments, même lorsqu’ils ont des tailles différentes.

Il est très utile pour créer des grilles, centrer des blocs, ou adapter une disposition selon la taille de l’écran.

✨ Exemple :

  justify-content: space-around; /* Espace égal entre les éléments */
  align-items: center; /* Aligne les éléments verticalement */
}

3. Les variables : let, const et var

  • let : variable réaffectable avec portée de bloc
  • const : constante, valeur fixe
  • var : ancien, portée fonction, à éviter

💡 let VS var

Portée : var est visible dans toute la fonction, let seulement dans le bloc courant.


// var (visible partout dans la fonction)
function testVar() {
  if (true) { var x = 10; }
  console.log(x); // ✅ 10
}
// let (visible seulement dans le bloc)
function testLet() {
  if (true) { let y = 20; }
  console.log(y); // ❌ Erreur ! y introuvable
}

Redéclaration : var autorise deux fois le même nom, let non : let a = 1; puis let a = 2; provoque une erreur.

Caractéristiquevarlet
PortéeFonctionnelleBloc
Redéclaration✅ Oui❌ Non
Hoisting✅ Oui⚠️ Oui mais non utilisable avant
Recommandé❌ Ancien JS✅ ES6+

Conclusion : Préfère let et const, oublie var.

7. 🧠 Méthodes utiles selon le type de données

📜 String (chaîne de caractères)

Les chaînes de caractères sont des textes. Voici les méthodes les plus utilisées :

  • str.length : retourne le nombre de caractères de la chaîne.
  • str.trim() : enlève les espaces en début et en fin de chaîne.
  • str.toUpperCase() : transforme tout le texte en MAJUSCULES.
  • str.toLowerCase() : transforme tout en minuscules.
  • str.includes("mot") : vérifie si un mot est présent dans la chaîne → true ou false.
  • str.replace("ancien", "nouveau") : remplace un mot par un autre.
  • str.split("") : coupe la chaîne en tableau de caractères.
const message = "  JavaScript est génial  ";
console.log(message.length);           // 27 (espaces inclus)
console.log(message.trim());           // "JavaScript est génial"
console.log(message.toUpperCase());    // "  JAVASCRIPT EST GÉNIAL  "
console.log(message.includes("génial")); // true
---

🔢 Number (nombres)

Voici les fonctions les plus utilisées pour manipuler les nombres :

  • n.toString() : convertit un nombre en chaîne de caractères.
  • parseInt(str) : convertit une chaîne en nombre entier.
  • parseFloat(str) : convertit une chaîne en nombre décimal.
  • Math.round(n) : arrondit à l’entier le plus proche.
  • Math.floor(n) : arrondit vers le bas.
  • Math.ceil(n) : arrondit vers le haut.
  • Math.random() : retourne un nombre aléatoire entre 0 et 1.

✅ Boolean (valeurs vraies ou fausses)

Les booléens représentent la vérité (true ou false). Voici des méthodes utiles :

  • Boolean(val) : convertit une valeur en booléen (ex : 0 → false, "abc" → true).

8. 🔧 Fonctions en JavaScript

Définition

Une fonction est un bloc de code réutilisable, qui effectue une tâche précise. Elle peut prendre des paramètres et retourner une valeur.

Synthaxe de base

function nomDeLaFonction(param1, param2) {
  // instructions
  return resultat; // facultatif
}

Exemple simple

function addition(a, b) {
  return a + b;
}

console.log(addition(3, 5)); // affiche 8

Ici, addition est une fonction qui prend deux paramètres a et b, et retourne leur somme.

Fonction sans paramètre et sans retour

function direBonjour() {
  console.log("Bonjour !");
}

direBonjour(); // affiche "Bonjour !"

🔍 Exemple plus avancé : Tester un palindrome

// Fonction qui vérifie si une chaîne est un palindrome
function isPalindrome(str) {
  // Nettoyage : on supprime tout sauf lettres et chiffres
  // /[^a-zA-Z0-9]/g : expression régulière
  // ^ : négation → on exclut ce qui est entre [ ]
  // a-zA-Z0-9 : garde lettres et chiffres uniquement
  // g : global → appliqué à toute la chaîne
  var cleaned = str.replace(/[^a-zA-Z0-9]/g, "").toLowerCase();

  // Inversion de la chaîne nettoyée
  var reversed = cleaned.split("").reverse().join("");

  // Comparaison de la chaîne nettoyée avec sa version inversée
  return cleaned === reversed;
}

// Exemple avec une célèbre phrase palindrome
var input = "A man, a plan, a canal: Panama";

// Affichage du résultat dans un élément HTML
document.getElementById("demo").innerHTML =
  isPalindrome(input) ? "C'est un palindrome." : "Ce n'est pas un palindrome.";

Ce code définit une fonction qui supprime les caractères non alphanumériques, met tout en minuscules, puis compare la chaîne avec sa version inversée. Grâce à l'expression régulière /[^a-zA-Z0-9]/g, seuls les lettres et chiffres sont conservés.

9. 🧱 Les structures de données : Objets en JavaScript

Définition

En JavaScript, les objets sont des structures de données permettant de regrouper plusieurs informations sous une seule variable. C’est l’équivalent d’une struct en langage C ou d’un dict en Python.

📦 Exemple : Utiliser une structure (objet)

<script>
// Déclaration d’un objet représentant une personne
const person = {
  firstname: "John",
  lastname: "Doe",
  age: 50,
  eyecolor: "blue"
};

// Affichage d’une phrase en utilisant les propriétés de l’objet
document.getElementById("demo").innerHTML =
  person.firstname + " is " + person.age + " years old.";
</script>

Remarque : en JavaScript, un objet comme person est une vraie structure de données :

  • Il regroupe plusieurs informations liées entre elles (nom, âge, etc.)
  • Chaque donnée est une clé : valeur (ex : firstname: "John")
  • C’est l’équivalent d’une struct en langage C ou d’un dict en Python
  • On accède aux valeurs avec la notation objet.propriété (ex : person.age)

10. 📚 Tableaux (Arrays) en JavaScript

🔎 Définition

Un tableau (array) est une structure qui permet de stocker plusieurs valeurs dans une seule variable. Chaque élément a une position (appelée indice), qui commence à 0. 

let fruits = ["Pomme", "Banane", "Orange"];
console.log(fruits[0]); // "Pomme"
console.log(fruits.length); // 3

🛠️ Fonctions utiles pour manipuler un tableau

  • array.length : retourne la taille du tableau.
  • array.push(element) : ajoute un élément à la fin.
  • array.pop() : supprime le dernier élément.
  • array.shift() : supprime le premier élément.
  • array.unshift(element) : ajoute un élément au début.
  • array.indexOf(valeur) : retourne l’indice de la valeur, ou -1 si non trouvée.
  • array.includes(valeur) : retourne true si la valeur est présente.
  • array.join(", ") : transforme le tableau en chaîne de caractères.
  • array.reverse() : inverse l’ordre du tableau.
  • array.sort() : trie les éléments (alphabétique ou numérique).

💡 Exemple complet

let nombres = [5, 3, 9, 1];

nombres.push(7);            // ajoute 7 → [5, 3, 9, 1, 7]
nombres.sort();             // trie → [1, 3, 5, 7, 9]
nombres.reverse();          // inverse → [9, 7, 5, 3, 1]
nombres.pop();              // supprime 1 → [9, 7, 5, 3]

console.log(nombres);       // affiche [9, 7, 5, 3]
console.log(nombres.includes(5));  // true
console.log(nombres.indexOf(7));   // 1

11. 🔁 Boucles (Loops) en JavaScript

Définition

Une boucle permet de répéter automatiquement un bloc de code, tant qu'une condition est vraie. Cela évite de réécrire plusieurs fois les mêmes instructions.

📌 Boucle for

Utilisée quand on sait combien de fois on veut répéter une tâche :

for (let i = 0; i < 5; i++) {
  console.log("i =", i);  // Affiche les nombres de 0 à 4
}

📌 Boucle while

Répète une instruction tant qu'une condition est vraie :

let i = 0;
while (i < 3) {
  console.log(i);  // Affiche 0, 1, 2
  i++;
}

📌 Boucle do...while

Exécute le code au moins une fois, puis vérifie la condition :

let i = 0;
do {
  console.log(i);  // Affiche 0, 1, 2
  i++;
} while (i < 3);

📌 Boucle for...of (tableaux)

Parcourt les éléments d’un tableau ou d’une chaîne :

let fruits = ["🍎", "🍌", "🍊"];
for (let fruit of fruits) {
  console.log(fruit);  // Affiche chaque fruit
}

📌 Boucle for...in (objets)

Parcourt les propriétés d’un objet :

let personne = { nom: "Alice", age: 25 };
for (let cle in personne) {
  console.log(cle + " → " + personne[cle]);  // Affiche nom → Alice, age → 25
}

💡 Exemple complet : moyenne d’un tableau

let notes = [15, 12, 17, 9];
let total = 0;

for (let note of notes) {
  total += note;  // Additionne toutes les notes
}

let moyenne = total / notes.length;
console.log("Moyenne =", moyenne);  // Affiche : Moyenne = 13.25

12. 🌳 Le DOM (Document Object Model)

Définition

Le DOM (Document Object Model) est une représentation sous forme d'arbre d’objets de la page HTML. JavaScript peut utiliser le DOM pour lire, modifier, ajouter ou supprimer des éléments HTML dynamiquement.

Exemple : le DOM transforme cette page HTML en structure navigable : 

html
 └── body
      └── p (id="demo")

🔧 Fonctions DOM utiles

  • document.getElementById("id") : sélectionne un élément par son identifiant
  • element.innerHTML = "..." : modifie le contenu HTML
  • element.style.color = "..." : change le style CSS
  • document.createElement("tag") : crée un nouvel élément HTML
  • element.appendChild(objet) : ajoute un élément dans un autre
  • element.remove() : supprime un élément

💡 Exemple : changer le contenu d’un paragraphe

<p id="demo">Bonjour</p>
<button onclick="changerTexte()">Changer</button>

<script>
function changerTexte() {
  document.getElementById("demo").innerHTML = "Texte mis à jour";
}
</script>

👉 Quand l'utilisateur clique sur le bouton, JavaScript accède à l'élément via le DOM et modifie son contenu.
